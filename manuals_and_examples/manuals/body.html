This is the documentation of rasdaman community, a free, open-source middleware extending standard relational database systems with the ability to efficiently query and manipulate multi-dimensional arrays of unlimited size. Such arrays are also called "raster data", "gridded data", etc. depending on the application domain.

  • If you want to set up rasdaman then you should consult the Installation and Administration Guide; best read in conjunction with the PostgreSQL External Products Guide.

  • Retrieval and manipulation of raster data is documented in the Query Language Guide (PDF). The visual query frontend, rview, is described in the rview guide (PDF).

  • For client API programming there is a C++ Development Guide (PDF) and Java Development Guide (PDF) together with documentation (HTML).

  • For those who wish to study the sources there is an HTML documentation of the complete system, API, and the PostgreSQL base DBMS interface

  • Finally, there is a list of error messages.
